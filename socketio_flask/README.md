# Socket.io + Flask

## Usage

```sh
python3 chatroom.py
# or
python3 -m chatroom
```

> http://localhost:5000/

### Docker

```sh
docker build . --tag chatroom_flask
docker run -p 5000:5000 chatroom_flask
```
